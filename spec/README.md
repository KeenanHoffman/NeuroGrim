# LSP Brains Specification

The LSP Brains specification has moved to its own repository:

**https://github.com/KeenanHoffman/LSP-Brains**

This directory previously contained the spec. As of v2, the specification and
the NeuroGrim implementation live in separate repositories:

- **LSP-Brains** — The methodology (spec, schemas, adoption guide)
- **NeuroGrim** — The product (Rust Brain engine, skills, domains)

The original spec files have been preserved in git history. The canonical versions
now live in the LSP-Brains repo.
